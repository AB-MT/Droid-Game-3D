Для создания дополнений не изменяйте уже созданные файлы .py, вы можете использовать их для ссылки на переменные.
Вам нужно создать новый файл py с классом и убедиться, что вы создали метод run (), который
имеет основной код для запуска
Если вам нужны другие файлы .py, вы можете создать папку со всеми файлами .py, включая основной файл.
Убедитесь, что вы создали папку, даже если вам нужен только один файл .py. Главный файл .py должен называться main.py.
Вы можете использовать базовую переменную для доступа ко всем переменным в основном классе.
